# scrapbox-tools

Personal libraries to develop scrapbox user scripts.

## CLI

### how to use

#### Deploy UserScript or UserCSS

```shell
$ export TOKEN=your_login_token
$ scrapbox-tools deploy project/page script.js
```

## Release

- publish command

  - `yarn release`

- unpublish command
  - `npm unpublish scrapbox-tools@$VERSION`
